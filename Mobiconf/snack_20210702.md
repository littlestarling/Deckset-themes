# **ウィスキーと割り物**
[.slidenumbers: false]

- 雑に理解するならこれで

| 飲み方 | 割り物 | 期待効果 |
| --- | --- | --- |
| ストレート | - | ウィスキー本来の香りと味を楽しみたい |
| ロック | 氷 | ヒエヒエ感を際立たせたい |
| 水割り</br> twice up | ウィスキーと同量の水 | 香りを際立たせて味わいたい |
| 水割り | 大体 1 : 2〜2.5くらい | じっくり味わいたい |
| ソーダ割り</br>a.k.a. ハイボール | 1 : 3〜4 くらい | 爽快感を出したい・癖のあるものの花を開かせたい |
| お湯割り | お湯 | 暖を取りたい・香りを一気に引き出したい |
